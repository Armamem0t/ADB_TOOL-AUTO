## 

### 无需任何另外Python;ADB环境 自带环境点击即可运行
-仅支持amd32/64版本的Windows10/11

## 功能
- 刷写分区 输入绝对路径
- 安装应用 支持绝对路径apk文件夹
- 多功能重启
- 查看手机信息
- 刷写官方刷机包 绝对路径选择
- 打开ADB命令行
